# 京东十大靶群三大人群矩阵校准

## 目的

校准森马京东十大靶群到 A/B/C 三大人群的映射矩阵，解决 v2.0.2 京东矩阵存在 uncovered 池导致合法十大靶群输入 `coverage < 1` 的结构问题。

本次校准目标是结构闭合：京东十大靶群每一行 `A+B+C=1`，当输入十大靶群合计为 1 时，输出 `coverage=1`，A/B/C raw 合计为 1。

## 使用方式

模型域实现入口：

```ts
import {
  estimateJdThreeAudienceShares,
  JD_RECOMMENDED_CALIBRATED_MATRIX,
} from "../apps/model/src/jd-three-audience-calibration.js";

const result = estimateJdThreeAudienceShares(segments, JD_RECOMMENDED_CALIBRATED_MATRIX);
```

冻结常量：

- `JD_THREE_AUDIENCE_CALIBRATION_VERSION = "semir_three_audience_v2.1.0-jd-calibrated"`
- `JD_RECOMMENDED_CALIBRATED_MATRIX`
- `JD_TEN_SEGMENT_LABELS`

后续七渠道算法实现必须消费上述冻结版本和矩阵；不得继续把 v2.0.2 京东 uncovered 池作为默认行为。

## 样例

以 `森马京东官旗26年1-6月` fixture 为例，十大靶群原始合计为 `1.0001`。v2.0.2 baseline 因矩阵 row sum 不完整，coverage 为 `66.29%`；目标校准矩阵输出 `A=20.72% / B=34.19% / C=45.10%`，coverage 为 `100.01%`。其中 `0.01pp` 来自源表四舍五入。

## 注意事项

- 四份已抽取 fixture 用于结构验证、before/after 和 2025/2026 年均目标校准；2024 年缺少原始十大靶群文件，只记录业务确认 A/B/C 目标。
- 本次校准不使用默认 expert prior，不保留京东 uncovered 池。
- 行为层信号仍不得回写份额层。
- 这是实质算法变化，版本升级为 `semir_three_audience_v2.1.0-jd-calibrated`。

候选 2：业务目标校准推荐。保留每行闭合与可复算约束，用 2025/2026 年真实十大靶群年均分布拟合用户确认的京东 A/B/C 目标。2024 年因缺少原始十大靶群文件，只作为业务目标记录，不参与矩阵拟合。

可复算推导：

1. 先把每份 fixture 的十大靶群归一化到合计 1，再分别求 2025、2026 年均分布。
2. 业务目标若 A/B/C 合计不等于 1，先归一化。2026 的 `21.1/34.7/44.1` 合计为 `99.9%`，归一化后为 `21.12/34.73/44.14`。
3. 固定非拟合行的先验权重：都市Z世代 `0.8/0/0.2`、都市中产 `0.2/0.8/0`、小镇青年 `0.1/0/0.9`、都市蓝领 `0.05/0/0.95`、银发一族 `0/0.3/0.7`、小镇中年 `0/0.1/0.9`。
4. 仅拟合四行的 A/B 权重：学生一族、都市家庭、小镇中产、小镇家庭；每行 C 由 `1-A-B` 得出。
5. 约束为 2025/2026 年均输出 A、B 精确等于归一化业务目标；C 因 row sum 闭合自动成立。
6. 由于两年 A/B 目标只有 4 个等式，不能唯一识别 10 行矩阵。tie-break 采用固定先验下的最小 L2 偏移：`min ||x - x_prior||^2` subject to `M x = y`，实现为 `x = x_prior + M^T (M M^T)^-1 (y - M x_prior)`。

## 风险

- 2024 年缺少原始十大靶群文件，当前只能记录目标，不能验证 2024 矩阵输出。
- 2025/2026 目标来自业务确认口径；若该口径未来变化，矩阵需要重新冻结版本。
- 推荐矩阵是目标校准结果，不应额外声明预测准确率。
- 两年年均 A/B 约束无法唯一识别 10 行矩阵；当前唯一性来自固定先验、可调行集合和最小 L2 偏移 tie-break。
- 由于取消 uncovered，coverage 不再表达京东矩阵未解释比例；后续如果还需要不确定性，应另设置信度或质量标记，而不是复用 uncovered。
